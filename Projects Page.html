Knowledge Is Power

Tools Used: Unreal Engine, Blender, SQLite, C++

Knowledge is Power is a roguelike dungeon crawler that aims to help students study by rewarding them with different upgrades and power-ups without punishing them for mistakes.
This study tool utilizes how addicting video games can be to encourage students to study.
This was my capstone project for my 491 course, and I ran into several problems during development.
A major issue was working with Unreal and its blueprint system. Eventually, through trial and error, we resolved it.
Another challenge was running the AI system alongside Unreal's limited use of C++. It took a lot of time, but we eventually figured it out.
This was a very fun project and the largest scale project I have worked on at Fullerton.

CrossRoads

Tools Used: Unreal Engine, MagickaVoxel

A recreation of the popular game Crossy Roads.
Collect coins while jumping across a field and see how high your score can get.
Currently, it has 8 different terrains generated every 10 jumps.
The main challenge was implementing the randomly generated terrain correctly and creating all the assets myself.
However, I learned a lot about Unreal Engine and the process of making my own game assets.

Discord Server Launcher

Tools Used: Python, Discord

Allows .bat files to be run from a Discord bot.
This tool fills in the commented-out Python script to run in specific servers or channels.
The Minecraft server is confirmed running and requires the file to be in the same directory as the server.
The main challenge with this project was interacting with Windows Powershell and working with the correct directories.
Overall, it was a fun project that I needed personally and now use frequently in my group's Discord.
